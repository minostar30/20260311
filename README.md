# 로또 번호 추천 서비스

한국 로또 6/45 번호를 추천해주는 웹 서비스입니다.

## 기능

- **랜덤 추천**: 1~45 중 6개 + 보너스 번호 추천
- **추천 이력**: 최근 5회 추천 번호 저장
- **2026년 1등 당첨번호**: 최신 회차 당첨번호 표시
- **데이터 분석 추천**: 역대 100회 1등 당첨 데이터 기반 3가지 추천
  - 핫 넘버 (최다 출현 번호)
  - 회귀 추천 (미출현 번호)
  - 균형 추천 (핫·콜드 혼합)

## 사용 방법

`lotto.html` 파일을 브라우저에서 열면 됩니다. 별도 서버 설치 불필요.
